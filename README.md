## Overview
Interactive dashboard analyzing employee attrition, engagement, tenure, and training effectiveness.

## Dataset
- 3,000+ employees (synthetic or anonymized)
- Key columns: Employee Name, Department, Title, Tenure, Engagement, Satisfaction, Training Program, Training Outcome, IsActive

## Key Insights
- Departments with highest attrition trends
- Engagement vs satisfaction highlights at-risk employees
- Training programs cost vs effectiveness
- Tenure and demographic patterns for workforce stability

## Tools & Skills
- Power BI Service & Desktop
- Power Query (Data Cleaning)
- DAX Calculations (KPIs: Attrition Rate, Average Tenure, Average Engagement)
- Interactive dashboard design & data storytelling
